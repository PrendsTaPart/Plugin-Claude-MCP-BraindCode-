# Tendances marché (Google Trends via DataForSEO + TikTok best-effort)

Détecte ce qui monte pour nourrir le calendrier éditorial. **De RECHERCHE** (pas les
tendances créatives de plateforme — voir Anti-collision).

## Étape 0
`reference/garde-fous-seo.md` (coûts, best-effort). Thèmes du client
`./rapido-kb/seo/seo-cibles.md` (ex. CRM PME, digitalisation restaurant, incubation startup).

## Sense
- **Google Trends** via DataForSEO (endpoint Trends de la famille `keywords_data`) :
  **saisonnalité** et **requêtes montantes** sur les thèmes du client. Coût annoncé.
- **TikTok (best-effort)** : tendances organiques via source disponible (Creative
  Center / veille). **Dire clairement quand la donnée n'est pas accessible** — pas
  d'invention.

## Plan
- **3 opportunités de contenu datées** : sujet, pourquoi maintenant (courbe/pic),
  format suggéré, fenêtre de publication.

## Act
- Pousser les 3 opportunités vers **`rapidocms:calendrier-editorial`** (proposées,
  planification confirmée).

## Anti-collision
- **`rapido-tiktok-ads:tendances-creatives-tiktok`** (à venir) = tendances
  **CRÉATIVES** de plateforme (formats, hooks). **Moi = tendances de RECHERCHE**
  (mots-clés, saisonnalité). Règle documentée des deux côtés.

## Garde-fous
Coût DataForSEO annoncé ; **TikTok best-effort** (dire si indispo) ; opportunités
**datées** et sourcées ; planification via calendrier-editorial **confirmée**.
