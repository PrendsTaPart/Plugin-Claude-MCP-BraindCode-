## LinkedIn — révisé 2026-07-15

- **Format** : texte natif ; carrousels **PDF** (documents) ; vidéo native ; sondages.
- **Longueur** : post **900-1 300 caractères** ; **hook 1-2 lignes** avant le « …voir plus ».
- **Structure gagnante** : hook fort → une **idée par ligne** (aération) → storytelling
  **pro** (leçon, chiffre, prise de position) → **CTA conversationnel** (une question).
- **Hashtags** : **3-5 max**, en fin de post.
- **Signaux algo** : **pas de lien externe dans le corps** (le mettre en **commentaire**) ;
  temps de lecture (dwell) et commentaires > likes ; poster en fenêtre **B2B** (matin
  semaine) ; répondre aux commentaires dans l'heure.
- **Erreurs qui tuent la portée** : lien dans le corps, pavé non aéré, hook faible,
  engagement-bait grossier, hashtags en rafale.
- **Funnel** : TOFU (prise de position, storytelling) → MOFU (cas, méthode) → BOFU
  (preuve client, offre douce en commentaire).

## Facebook — révisé 2026-07-15

- **Format** : texte court, **vidéo carrée/verticale**, photos natives, **événements**,
  **groupes** (local — résonne pour les restaurateurs FoodEatUp).
- **Longueur** : **40-120 mots** (court).
- **Structure** : accroche concrète → bénéfice local/immédiat → **CTA clic doux**.
- **Signaux algo** : **natif > lien** (un lien réduit la portée ; le mettre en commentaire
  si besoin) ; vidéo native favorisée ; interactions **de proximité** (groupes, local).
- **Erreurs** : lien en tête, ton corporate, texte trop long, pas d'ancrage local.
- **Funnel** : TOFU (contenu local/utile) → MOFU (offre, événement) → BOFU (promo, avis).

## Instagram — révisé 2026-07-15

- **Format** : **caption** (feed), **carrousels 6-10 slides** à valeur, **Reels** (vidéo
  verticale), **Stories** (sondages/questions), lien en **bio/sticker**.
- **Longueur** : caption = **hook dans les 125 premiers caractères** (avant le « plus ») ;
  Reel = caption **courte** + **texte à l'écran** ; carrousel = 1 idée/slide.
- **Structure** : hook visuel + verbal → valeur slide par slide (carrousel) → **CTA**
  (enregistrer/partager/commenter) ; Stories = interaction (sticker sondage/question).
- **Hashtags** : **3-8 mixtes** (niche + volume).
- **Signaux algo** : **saves** et **shares** > likes ; watch-time des Reels ; complétion
  de carrousel. **Pas de lien cliquable** dans la caption (bio/sticker).
- **Erreurs** : hook noyé après 125 car., carrousel sans valeur, Reel sans texte à
  l'écran, hashtags hors sujet.
- **Funnel** : TOFU (Reels découverte) → MOFU (carrousels méthode) → BOFU (témoignages,
  lien bio).

## TikTok — révisé 2026-07-15

- **Le SCRIPT prime** (pas la caption). Livrable = **script de tournage**.
- **Structure gagnante** : **hook visuel + verbal < 3 s** → **boucle ouverte** → valeur →
  **payoff** → CTA. Durée **15-45 s**. **Sous-titres obligatoires**. **Son natif** (tendance).
- **Ton** : **créateur** (jamais corporate). Texte à l'écran horodaté.
- **Hashtags** : **3-5** dont au moins un **niche**.
- **Signaux algo** : **rétention** (les 3 premières s décident) + **complétion** + rewatch ;
  partages et commentaires ; son tendance.
- **Erreurs** : intro lente, ton corporate, pas de sous-titres, hook « bonjour à tous »,
  durée trop longue sans payoff.
- **Funnel** : quasi tout **TOFU** (découverte) ; MOFU/BOFU via séries et lien bio.

## Déclinaison inter-réseaux (rappel)

Une idée-noyau → **4 déclinaisons NATIVES** (une par grammaire), **jamais** un
copier-coller raccourci. Voir `declinaison-multi-reseaux` (CW3). Les Reels IG et Shorts
YouTube réutilisent le **script TikTok** avec la grammaire ajustée.
