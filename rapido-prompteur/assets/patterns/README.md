# Patterns de prompt — par usage

Bibliothèque de **structures de prompt francisées** (placeholders `[…]`), classées
par usage. Chaque fiche indique les **moteurs compatibles** et sa **source + licence**
en pied. Toutes excluent les IP tierces et les styles d'artistes vivants
(cf. `../../reference/regles-de-construction.md`).

| Fiche | Usage | Média |
|---|---|---|
| `packshot.md` | Photo produit e-commerce/pub | image |
| `portrait.md` | Portrait pro / équipe / avatar | image |
| `food.md` | Plat / boisson appétissant (FoodEatUp) | image |
| `ugc-ad.md` | UGC / pub authentique | vidéo (ou image) |
| `cinematique.md` | Plan cinématographique | vidéo / still |
| `personnage.md` | Personnage/mascotte cohérent(e) | image → vidéo |
| `hooks-viraux.md` | Accroche 0-2 s (TikTok/Reels) | vidéo |
| `web-apps.md` | Brief landing/app (Lovable mode B) | web / app |

> **Rappel** : les valeurs concrètes (ratios, résolutions, durées) ne sont **pas**
> dans ces fiches — elles se **lisent en live** sur le moteur cible
> (`../../reference/grammaire-des-moteurs.md`).
