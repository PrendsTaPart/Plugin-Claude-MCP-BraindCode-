# rapido-canva — Design Canva alimenté par les données Rapido

Design Canva alimenté par les données Rapido : menus restaurant (FoodEatUp), visuels sociaux (RapidoCMS), supports commerciaux (RapidoCRM), présentations CODIR (4 serveurs).

## Skills (7)

| Skill | Quand l'utiliser |
|---|---|
| `brand-guidelines-anthropic` | Explicitement le look Anthropic (couleurs et typographies officielles) pour un artefact |
| `canvas-design` | Un poster, une œuvre visuelle ou un design statique original en .png ou .pdf. Philosophie de d… |
| `menu-restaurant-design` | Créer son menu, un menu imprimable ou une carte à imprimer pour son restaurant |
| `presentation-codir` | La présentation du CODIR ou des slides pour le comité de direction |
| `supports-commerciaux` | Une proposition commerciale, une présentation de vente ou une carte de visite |
| `theme-factory` | Appliquer un thème (couleurs, typographies) à un artefact |
| `visuels-sociaux-canva` | Un visuel Canva, un design pour Instagram/LinkedIn/Facebook ou un template de post |

## Agents (1)

- **`studio-creatif`** — Directeur de studio créatif multi-canaux.
