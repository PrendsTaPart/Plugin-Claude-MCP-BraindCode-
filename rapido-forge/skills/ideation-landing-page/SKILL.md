# Landing Page

**Catégorie** : Idéation  
**Durée** : 25-35 min

## Pourquoi

La section Hero est ce que 100% des visiteurs voient en premier. Tu as 3 secondes pour capter leur attention. Un bon Hero peut doubler ton taux de conversion.

## Objectif

Rédiger un titre et sous-titre Hero accrocheurs qui convertissent.

## Livrable attendu

H1 (titre principal) + H2 (sous-titre) + CTA button text

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Lister les bénéfices clés** — Qu'est-ce que le client gagne concrètement ?
2. **Rédiger 10 variations H1** — Formules : 'Get X without Y', 'The #1 way to...', 'Finally, X'
   > Prompt: Écris 10 titres Hero pour [PRODUIT] qui aide [PERSONA] à [BÉNÉFICE]. Ton: [VIBE]
3. **Rédiger les sous-titres** — Précise le bénéfice ou la méthode
4. **Créer le CTA** — Action claire : 'Commencer gratuitement', 'Réserver une démo'
5. **A/B tester** — Prépare 2-3 versions à tester

## Pro tips

- Le titre doit être compréhensible sans le sous-titre
- Inclus des chiffres quand c'est possible ('En 7 jours')
- Le CTA doit promettre une action, pas un engagement

## Erreurs fréquentes

- Titre trop vague ou corporate
- Parler de features au lieu de bénéfices
- CTA sans urgence ou valeur claire

## Données & serveurs MCP

- **RapidoCMS** (`rapidocms`) — données réelles, lecture d'abord
- **Lovable** (`lovable`, via le plugin `rapido-lovable`) pour la construction réelle

## Livrable — toujours dans la KB du client

Écrire le livrable daté dans
`./rapido-kb/startup/forge/ideation/ideation-landing-page.md` (le dossier du client,
dans SON répertoire de travail — jamais dans le plugin). Si des chiffres
clés en sortent, proposer de mettre à jour
`./rapido-kb/startup/business-plan/hypotheses.md` (source + date).
