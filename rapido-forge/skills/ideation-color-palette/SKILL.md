# Color Palette

**Catégorie** : Idéation  
**Durée** : 20-30 min

## Pourquoi

Les couleurs évoquent des émotions et influencent les décisions d'achat. 85% des consommateurs citent la couleur comme raison principale de leur achat. Ta palette définit l'ambiance de ta marque.

## Objectif

Définir une palette de couleurs cohérente et professionnelle.

## Livrable attendu

Palette de 5 couleurs avec codes HEX, significations et règles d'utilisation

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Analyser la psychologie des couleurs** — Bleu = confiance, Rouge = urgence, Vert = nature...
2. **Définir la couleur principale** — Celle qui représente le plus ta marque
3. **Créer les couleurs secondaires** — 2-3 couleurs complémentaires
4. **Ajouter les neutres** — Blanc, gris, noir pour le texte et les fonds
5. **Tester l'accessibilité** — Contraste suffisant pour les textes

## Pro tips

- La règle 60-30-10 : 60% couleur dominante, 30% secondaire, 10% accent
- Teste ta palette sur différents écrans
- Vérifie le contraste pour l'accessibilité (WCAG)

## Erreurs fréquentes

- Trop de couleurs (5 max)
- Couleurs qui ne s'accordent pas
- Ignorer l'accessibilité pour les daltoniens
