# Tiktok Ads Creator

**Catégorie** : Scale  
**Durée** : 60 min

## Pourquoi

TikTok a le meilleur engagement et les CPM les plus bas. Les Ads natives (qui ressemblent à du contenu) performent 10x mieux que les pubs traditionnelles. Idéal pour les 18-35 ans et le D2C.

## Objectif

Créer des publicités TikTok natives qui ne ressemblent pas à des pubs.

## Livrable attendu

3 créatives TikTok natives + campagne configurée

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Étudier le Creative Center** — Analyse les top ads de ta catégorie sur TikTok
2. **Scripter les hooks** — 3 secondes max pour capturer l'attention
   > Prompt: Quels hooks TikTok pour une startup [SECTEUR] ciblant [AUDIENCE] ?
3. **Créer en format natif** — Vertical, texte intégré, voix off authentique
4. **Utiliser les trends** — Sons populaires, effets tendance, formats viraux
5. **A/B tester les hooks** — Même contenu, hooks différents

## Pro tips

- Don't make ads, make TikToks
- Les UGC performent mieux que le contenu brandé
- Hook in 1 second, story in 15, CTA at the end
- Utilise Spark Ads pour booster des posts organiques

## Erreurs fréquentes

- Faire des pubs TV sur TikTok
- Ignorer les trends et sons populaires
- Format horizontal ou trop long
