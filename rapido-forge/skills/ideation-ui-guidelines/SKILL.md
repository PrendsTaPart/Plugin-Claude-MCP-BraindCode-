# Ui Guidelines

**Catégorie** : Idéation  
**Durée** : 20-25 min

## Pourquoi

La typographie représente 95% du design web. Le choix de tes fonts impacte la lisibilité, la personnalité de ta marque et l'expérience utilisateur globale.

## Objectif

Sélectionner une combinaison de fonts harmonieuse et lisible.

## Livrable attendu

Paire de fonts (display + body) avec tailles et règles d'utilisation

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Définir la personnalité** — Sérieux, fun, moderne, classique ?
2. **Choisir le font display** — Pour les titres - plus expressif
3. **Choisir le font body** — Pour le texte - très lisible
4. **Tester la combinaison** — Visualise sur différentes tailles
5. **Documenter l'échelle** — H1, H2, H3, body, caption...

## Pro tips

- Limite-toi à 2 fonts maximum
- Combine serif + sans-serif pour du contraste
- Teste la lisibilité sur mobile

## Erreurs fréquentes

- Fonts trop décoratives pour le body text
- Trop de variations (italique, bold, light...)
- Ignorer la disponibilité gratuite (Google Fonts)
