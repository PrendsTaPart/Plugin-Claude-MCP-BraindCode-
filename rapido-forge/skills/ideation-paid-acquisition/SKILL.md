# Paid Acquisition

**Catégorie** : Idéation  
**Durée** : 60-90 min

## Pourquoi

La publicité payante permet de scaler l'acquisition rapidement une fois que tu as validé ton product-market fit.

## Objectif

Élaborer une stratégie d'acquisition payante.

## Livrable attendu

Plan média avec budget et ROI cible

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Définir le budget** — CAC cible × clients souhaités
2. **Choisir les canaux** — Meta, Google, LinkedIn selon cible
3. **Créer les audiences** — Lookalike, retargeting, intérêts
4. **Planifier les créatives** — 3-5 variations pour A/B test
5. **Définir les KPIs** — CAC, ROAS, LTV/CAC

## Pro tips

- Commence avec un petit budget pour tester
- Le retargeting a le meilleur ROAS
- Teste les créatives avant de scaler le budget

## Erreurs fréquentes

- Scaler trop vite sans data
- Une seule créative sans A/B test
- Ignorer le tracking des conversions

## Données & serveurs MCP

- **RapidoCRM** (`rapidocrm`) — données réelles, lecture d'abord
- **Facebook ads** (`facebook-ads`, via le plugin `rapido-meta-ads`) — ⚠️ argent réel : tout se crée en PAUSED, activation après accord explicite
