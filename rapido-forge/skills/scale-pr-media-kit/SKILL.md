# Pr Media Kit

**Catégorie** : Scale  
**Durée** : 45 min

## Pourquoi

Un Media Kit professionnel facilite la couverture presse et donne de la crédibilité. Les journalistes utilisent directement les assets fournis - rends-leur la vie facile.

## Objectif

Créer un Media Kit complet prêt à envoyer aux journalistes et partenaires.

## Livrable attendu

Media Kit PDF + dossier assets (logos, photos, stats)

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Rédiger la bio company** — 3 versions : 1 phrase, 1 paragraphe, 1 page
2. **Préparer les visuels** — Logo (PNG, SVG), photos fondateurs, screenshots produit
   > Prompt: Quels éléments essentiels inclure dans un Press Kit pour une startup [SECTEUR] ?
3. **Compiler les stats clés** — Traction, funding, clients notables, growth
4. **Créer les bios fondateurs** — Photo + bio + LinkedIn + citation
5. **Assembler le kit** — PDF téléchargeable + lien Drive assets HD

## Pro tips

- Inclus des citations quotables que les journalistes peuvent reprendre
- Formats variés des logos (dark/light, horizontal/vertical)
- Mets à jour le kit à chaque milestone majeur

## Erreurs fréquentes

- Visuels basse résolution inutilisables pour la presse
- Informations datées (funding annoncé il y a 2 ans)
- Kit trop lourd à télécharger

## Données & serveurs MCP

- **RapidoCRM** (`rapidocrm`) — données réelles, lecture d'abord

## Livrable — toujours dans la KB du client

Écrire le livrable daté dans
`./rapido-kb/startup/forge/scale/scale-pr-media-kit.md` (le dossier du client,

clés en sortent, proposer de mettre à jour
`./rapido-kb/startup/business-plan/hypotheses.md` (source + date).
