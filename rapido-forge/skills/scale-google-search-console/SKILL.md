# Google Search Console

**Catégorie** : Scale  
**Durée** : 45 min

## Pourquoi

Google Search Console te montre exactement comment Google voit ton site. C'est l'outil gratuit le plus puissant pour diagnostiquer tes problèmes SEO et découvrir les requêtes qui amènent du trafic.

## Objectif

Configurer GSC et analyser les premières données de performance SEO.

## Livrable attendu

GSC configuré + sitemap soumis + premier rapport de performance

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Ajouter la propriété** — Ajoute ton domaine dans Search Console et vérifie la propriété
2. **Soumettre le sitemap** — Génère et soumets ton sitemap.xml
   > Prompt: Comment générer un sitemap pour [TECHNOLOGIE] ?
3. **Vérifier l'indexation** — Utilise l'inspection d'URL pour vérifier les pages importantes
4. **Analyser les performances** — Identifie les requêtes top, les pages populaires, le CTR
5. **Corriger les erreurs** — Traite les erreurs d'indexation et les problèmes signalés

## Pro tips

- Vérifie les Core Web Vitals pour l'UX
- Utilise l'Inspection d'URL pour demander un réindexation rapide
- Exporte les requêtes vers un sheet pour analyse approfondie

## Erreurs fréquentes

- Ignorer les erreurs de couverture
- Ne pas soumettre de sitemap
- Oublier les versions www et non-www
