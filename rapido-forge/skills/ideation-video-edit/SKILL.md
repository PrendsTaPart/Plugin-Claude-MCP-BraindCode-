# Video Edit

**Catégorie** : Idéation  
**Durée** : 35-45 min

## Pourquoi

Un teaser vidéo capte l'attention en quelques secondes. C'est parfait pour les réseaux sociaux et le lancement.

## Objectif

Créer une vidéo teaser courte et percutante.

## Livrable attendu

Teaser de 15-60 secondes optimisé pour les réseaux

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Sélectionner les best moments** — Clips les plus impactants
2. **Créer le hook visuel** — 3 premières secondes captivantes
3. **Monter le teaser** — Rythme rapide, transitions dynamiques
4. **Ajouter musique et texte** — Sous-titres, call-out, CTA final
5. **Exporter pour chaque plateforme** — Ratios: 9:16, 1:1, 16:9

## Pro tips

- Les 3 premières secondes déterminent tout
- Musique énergique = plus d'engagement
- Sous-titres obligatoires (80% regardent sans son)

## Erreurs fréquentes

- Teaser trop long ou lent
- Pas de sous-titres
- CTA absent ou flou

## Données & serveurs MCP

- **RapidoCRM** (`rapidocrm`) — données réelles, lecture d'abord
- **HyperFrames by HeyGen** (`hyperframes`) — le rendu MP4 est une action payante : validation avant

## Livrable — toujours dans la KB du client

Écrire le livrable daté dans
`./rapido-kb/startup/forge/ideation/ideation-video-edit.md` (le dossier du client,
dans SON répertoire de travail — jamais dans le plugin). Si des chiffres
clés en sortent, proposer de mettre à jour
`./rapido-kb/startup/business-plan/hypotheses.md` (source + date).
