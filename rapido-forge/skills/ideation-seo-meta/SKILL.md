# Seo Meta

**Catégorie** : Idéation  
**Durée** : 30-40 min

## Pourquoi

53% du trafic web vient du SEO organique. Optimiser tes méta-données dès le lancement te positionne pour un trafic gratuit à long terme.

## Objectif

Optimiser le SEO de base de ton site pour les moteurs de recherche.

## Livrable attendu

Meta titles, descriptions, keywords pour toutes les pages

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Rechercher les mots-clés** — 3-5 mots-clés principaux + longue traîne
2. **Rédiger les meta titles** — 50-60 caractères, mot-clé au début
3. **Rédiger les meta descriptions** — 150-160 caractères, CTA inclus
4. **Optimiser les images** — Alt text descriptifs avec mots-clés
5. **Vérifier le technique** — Sitemap XML, robots.txt, vitesse

## Pro tips

- Chaque page doit cibler un mot-clé unique
- Inclus le nom de ta marque dans les titles
- Utilise Google Search Console dès le lancement

## Erreurs fréquentes

- Mêmes meta titles sur toutes les pages
- Keyword stuffing (trop de mots-clés)
- Ignorer la vitesse de chargement

## Données & serveurs MCP
