# Customer Journey

**Catégorie** : Scale  
**Durée** : 60 min

## Pourquoi

Le Customer Journey Map révèle chaque point de contact et friction. Les entreprises qui mappent le journey ont 54% plus de ROI marketing car elles optimisent les bons moments.

## Objectif

Mapper les 7 étapes du parcours client de la découverte à l'advocacy.

## Livrable attendu

Customer Journey Map complet avec émotions, touchpoints et opportunités

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Définir les étapes** — Awareness → Consideration → Decision → Purchase → Retention → Advocacy
2. **Lister les touchpoints** — Chaque interaction client-marque par étape
   > Prompt: Liste tous les touchpoints d'un client [PERSONA] achetant [PRODUIT]
3. **Mapper les émotions** — Comment le client se sent à chaque étape ?
4. **Identifier les frictions** — Où sont les abandons, frustrations, confusions ?
5. **Prioriser les améliorations** — Quick wins vs investissements long terme

## Pro tips

- Base ton journey sur des données réelles (analytics, interviews), pas des suppositions
- Les moments 'wow' comptent autant que l'élimination des frictions
- Segmente les journeys par persona si les parcours diffèrent significativement

## Erreurs fréquentes

- Journey map trop linéaire (les vrais parcours sont chaotiques)
- Se concentrer uniquement sur le digital
- Oublier le post-achat qui détermine la rétention

## Données & serveurs MCP
