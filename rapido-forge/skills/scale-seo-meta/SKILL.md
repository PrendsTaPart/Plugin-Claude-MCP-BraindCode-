# Seo Meta

**Catégorie** : Scale  
**Durée** : 45 min

## Pourquoi

Les balises meta (title, description) sont ta première impression sur Google. Un bon titre augmente ton CTR de 20-30%. C'est du SEO gratuit et immédiat.

## Objectif

Optimiser les balises meta de toutes tes pages clés pour maximiser le CTR.

## Livrable attendu

Audit meta + nouvelles balises optimisées pour 10+ pages

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Auditer les meta actuelles** — Liste toutes tes pages et leurs meta actuelles
2. **Rechercher les mots-clés** — Identifie les keywords cibles pour chaque page
   > Prompt: Quels mots-clés cibler pour une page [TYPE] sur [SUJET] ?
3. **Écrire les titles** — Title < 60 caractères, mot-clé au début, incitation au clic
4. **Écrire les descriptions** — Description < 160 caractères, action, bénéfice, urgence
5. **Implémenter et vérifier** — Déploie et vérifie dans GSC

## Pro tips

- Utilise des power words : Gratuit, Guide, 2024, Meilleur
- Inclus des chiffres quand pertinent
- Teste différentes versions et mesure le CTR

## Erreurs fréquentes

- Copier-coller la même description sur toutes les pages
- Dépasser les limites de caractères (troncature)
- Oublier le call-to-action dans la description
