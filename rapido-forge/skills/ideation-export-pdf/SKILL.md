# Export Pdf

**Catégorie** : Idéation  
**Durée** : 30-45 min

## Pourquoi

Un dossier startup complet est essentiel pour les investisseurs, banques et partenaires. C'est ta carte de visite professionnelle.

## Objectif

Compiler tous les documents clés dans un dossier PDF.

## Livrable attendu

Dossier PDF complet avec Executive Summary, BMC, financials, pitch deck

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Lister les documents** — Executive Summary, BMC, Lean Canvas, Pitch Deck, Financials
2. **Créer la page de couverture** — Logo, nom, tagline, contact
3. **Compiler les documents** — Ordre logique, pagination
4. **Ajouter le sommaire** — Liens cliquables vers chaque section
5. **Exporter en PDF** — Qualité haute, taille optimisée

## Pro tips

- Garde le fichier < 10MB pour l'envoi par email
- Ajoute des liens vers ton site et réseaux
- Mets à jour régulièrement avec les nouvelles métriques

## Erreurs fréquentes

- Documents non mis à jour
- Fichier trop lourd
- Pas de contact visible

## Données & serveurs MCP

- **RapidoCRM** (`rapidocrm`) — données réelles, lecture d'abord

## Livrable — toujours dans la KB du client

Écrire le livrable daté dans
`./rapido-kb/startup/forge/ideation/ideation-export-pdf.md` (le dossier du client,
dans SON répertoire de travail — jamais dans le plugin). Si des chiffres
clés en sortent, proposer de mettre à jour
`./rapido-kb/startup/business-plan/hypotheses.md` (source + date).
