# Semrush Audit

**Catégorie** : Scale  
**Durée** : 90 min

## Pourquoi

Semrush est l'outil SEO pro le plus complet. Un audit révèle tous tes problèmes techniques, tes opportunités de mots-clés et ce que font tes concurrents. C'est la base d'une stratégie SEO sérieuse.

## Objectif

Réaliser un audit SEO complet et créer un plan d'action priorisé.

## Livrable attendu

Audit technique + analyse concurrentielle + plan SEO 90 jours

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Site Audit** — Lance un audit technique complet de ton site
2. **Analyser les erreurs** — Priorise : critiques > warnings > notices
3. **Keyword Gap Analysis** — Compare tes keywords vs 3 concurrents
   > Prompt: Quels mots-clés mes concurrents [CONCURRENT1, CONCURRENT2] rankent que je n'ai pas ?
4. **Backlink Analysis** — Identifie les opportunités de liens
5. **Position Tracking** — Configure le suivi de tes mots-clés cibles
6. **Créer le plan d'action** — Quick wins + moyen terme + long terme

## Pro tips

- Focus sur les pages avec le plus de potentiel (position 5-20)
- Le Keyword Gap révèle des opportunités cachées
- Les backlinks de concurrents = prospects pour outreach
- Utilise le Content Template pour optimiser les pages

## Erreurs fréquentes

- Se perdre dans les métriques sans agir
- Ignorer les problèmes techniques (ralentissent tout)
- Ne pas tracker les positions dans le temps
