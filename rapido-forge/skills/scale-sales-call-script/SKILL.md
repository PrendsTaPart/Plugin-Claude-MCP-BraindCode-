# Sales Call Script

**Catégorie** : Scale  
**Durée** : 45 min

## Pourquoi

Un script de vente structuré te donne confiance et assure une couverture complète des points clés. Les commerciaux avec script closent 33% plus car ils ne ratent aucune étape.

## Objectif

Créer un script d'appel de vente structuré en 7 étapes avec transitions naturelles.

## Livrable attendu

Script d'appel complet + objection cards + guide de closing

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Structurer l'ouverture** — Accroche, agenda, temps estimé, permission
2. **Préparer la découverte** — Questions SPIN, écoute active, prise de notes
   > Prompt: Crée un script d'appel de vente en 7 étapes pour vendre [PRODUIT] à [PERSONA]
3. **Structurer la présentation** — Lien problème → solution, démo ciblée, cas clients
4. **Préparer le traitement d'objections** — Intégrer les réponses du playbook
5. **Préparer le closing** — Résumé valeur, prochaines étapes, engagement clair

## Pro tips

- Le script est un guide, pas un texte à lire mot pour mot
- Prépare des questions de secours si le client ne parle pas
- Le closing commence dès l'ouverture (obtenir micro-engagements)

## Erreurs fréquentes

- Lire le script de façon robotique
- Monopoliser la parole (ratio idéal 40/60)
- Oublier de confirmer les prochaines étapes

## Données & serveurs MCP

- **RapidoCRM** (`rapidocrm`) — données réelles, lecture d'abord

## Livrable — toujours dans la KB du client

Écrire le livrable daté dans
`./rapido-kb/startup/forge/scale/scale-sales-call-script.md` (le dossier du client,
dans SON répertoire de travail — jamais dans le plugin). Si des chiffres
clés en sortent, proposer de mettre à jour
`./rapido-kb/startup/business-plan/hypotheses.md` (source + date).
