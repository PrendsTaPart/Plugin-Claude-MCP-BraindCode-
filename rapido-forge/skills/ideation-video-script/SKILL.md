# Video Script

**Catégorie** : Idéation  
**Durée** : 35-45 min

## Pourquoi

La vidéo convertit 5x mieux que le texte. Un script bien structuré garantit une vidéo engageante qui maintient l'attention du spectateur.

## Objectif

Scripter une vidéo tutoriel ou explicative.

## Livrable attendu

Script complet avec intro, corps, CTA et indications visuelles

## Étape 0 — Contexte (obligatoire)

Charger `./rapido-kb/` — surtout `./rapido-kb/startup/` (vision, persona,
offre, hypothèses construits par `dossier-startup-360` et
`interview-business-plan`). La KB PRIME : ne jamais redemander une
information déjà validée, la reformuler pour confirmation. Les chiffres
réels viennent des MCP — jamais de mémoire ; tout chiffre sans source datée
porte la mention « hypothèse fondateur, confiance faible ».

## Étapes

1. **Définir l'objectif** — Éduquer, vendre, démontrer ?
2. **Structurer en 3 actes** — Hook (5s) → Promesse → Contenu → CTA
3. **Écrire le hook** — Question choc ou stat surprenante
   > Prompt: Écris 5 hooks vidéo de 5 secondes pour capter l'attention sur [SUJET]
4. **Développer le corps** — 3-5 points clés avec transitions
5. **Ajouter les indications** — [B-roll], [Graphique], [Texte à l'écran]

## Pro tips

- Parle comme tu parles, pas comme tu écris
- 1 minute de script ≈ 150 mots
- Les 3 premières secondes déterminent tout

## Erreurs fréquentes

- Script trop long et monotone
- Pas de variation visuelle
- Oublier le CTA final

## Données & serveurs MCP

- **RapidoCRM** (`rapidocrm`) — données réelles, lecture d'abord
- **HyperFrames by HeyGen** (`hyperframes`) — le rendu MP4 est une action payante : validation avant
