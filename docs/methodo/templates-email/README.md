# Templates email (français)

> Adaptations françaises **reformulées** des 3 templates de
> zubair-trabzada/ai-marketing-claude (MIT © 2026 Zubair Trabzada) — non-verbatim,
> voir `../ops/NOTICE.md`. Placeholders entre `[crochets]` à remplacer par les
> données réelles (jamais inventées). Exécution via RapidoCRM
> (`create_template_email` + `schedule_email`/`send_newsletter`), **chaque envoi
> confirmé**, désinscription présente, consentement RGPD.

- `bienvenue.md` — séquence de bienvenue (5 emails, J0→J8)
- `nurture.md` — séquence de nurture (6 emails, J1→J12)
- `lancement.md` — séquence de lancement (8 emails, J-7→J+7)

Cibles de performance (indicatives, à confirmer avec vos `benchmarks.md`) :
ouverture 40-60 %, clic 8-15 %, réponse 3-5 %, désinscription < 0,5 %.
