# Séquence de bienvenue (5 emails)

> `../ops/NOTICE.md`). Placeholders `[…]` à remplacer par des données réelles.

## Email 1 — immédiat
- **Objet** : Bienvenue ! Votre première victoire rapide est là
- **Aperçu** : Vous avez fait le bon choix…
- **Corps** : accueil chaleureux → ce à quoi s'attendre (3-4 puces) → une
  ressource actionnable tout de suite → **un seul CTA**.

## Email 2 — J+2
- **Objet** : La vraie raison pour laquelle [problème] arrive
- **Aperçu** : Ce n'est pas ce que vous croyez…
- **Corps** : problème → agitation → solution ; une preuve → CTA.

## Email 3 — J+4
- **Objet** : Comment [client] a obtenu [résultat précis]
- **Aperçu** : Et vous aussi…
- **Corps** : mini cas client (situation → action → résultat) → CTA.

## Email 4 — J+6
- **Objet** : « Je n'ai pas le temps pour [tâche] »
- **Aperçu** : Voici la version en 10 minutes…
- **Corps** : objection n°1 → recadrage → démarrage express → CTA.

## Email 5 — J+8
- **Objet** : Petite question sur votre expérience
- **Aperçu** : J'aimerais votre avis honnête.
- **Corps** : prise de contact → une question → proposition d'aide → CTA doux
  (répondre à l'email).
