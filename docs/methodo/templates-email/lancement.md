# Séquence de lancement (8 emails, 3 phases)

> Adaptation FR reformulée — source MIT © 2026 Zubair Trabzada (voir
> `../ops/NOTICE.md`). Fenêtre ~2 semaines. Placeholders `[…]` à remplacer par
> des données réelles.

## Phase pré-lancement
1. **J-7** — Objet : Quelque chose de grand arrive… (teaser)
2. **J-5** — Objet : Pourquoi [approche courante] vous fait défaut
3. **J-2** — Objet : Premier aperçu : [nom du produit] (sneak peek)

## Phase lancement
4. **J0 (matin)** — Objet : C'est en ligne : [nom du produit] est là
5. **J+1** — Objet : « [citation résultat] » — [nom du client] (preuve sociale)
6. **J+3** — Objet : Vos 5 questions sur [produit], nos réponses (FAQ/objections)

## Phase clôture
7. **J+5** — Objet : Plus que 48 h pour [tarif de lancement]
8. **J+7** — Objet : Dernier appel — [l'offre] expire ce soir

> Chaque email : objet + aperçu + corps (un angle, un CTA) + PS. Deadlines
> **réelles** uniquement. Envois planifiés et **confirmés** via RapidoCRM.
